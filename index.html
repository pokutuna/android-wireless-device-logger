Whats this?

This android app is a logger to record neighboring Bluetooth and WiFi devices for lifelogging and recognizing user contexts. And it's able to send log data to Dropbox by one tap.

  • it works on android 4.0~
  • checked the workings of this app on Nexus 7 (4.2.1)

How to Install

  1. In Android, check Settings > Security > Unknown sources
  2. Download wd-logger.apk from right side of this page and tap it!
